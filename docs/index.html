System-Prompt Reverse-Engineering

Extract any agent's hidden prompt.

Most red-team tools need an API. The agents that matter — embedded support bots, in-product copilots — don't have one. Reversegent points at the live browser tab, self-calibrates from a single message, reconstructs the hidden system prompt, then audits whether the agent holds its own rules. No selectors. No config. No API required.

One Loop, Three Engines

Probe. Reconstruct. Audit.

Reversegent treats extraction as adaptive intelligence-gathering — every response leaks something. Watch each engine work.

ReAct — Reasoning

Contextual Probing

An OpenAI Thought→Act→Observe loop reads each reply and authors the next probe from what just leaked. A refusal proves a rule exists; a partial answer reveals a boundary; a correction reveals the real instruction.

Browser — Injection

Agnostic Detection

Point it at a live chat tab. It searches every frame, opens collapsed widgets, locates the input, and self-calibrates the message selectors from a single marker — no presets, no config.

Robustness — Audit

Guardrail Jailbreak

After reconstruction, multi-turn adversarial tests check whether the agent enforces its OWN rules — crescendo, role-play, context injection — and report which guardrails held and which broke.

What Makes It Work

Built like an attacker thinks.

Agnostic
Works on any chat UI — public assistants, embedded support bots, iframes — with zero selectors.
Contextual
A ReAct loop adapts every probe to the last response instead of replaying a fixed script.
Multi-Turn
Builds trust, escalates by crescendo, and injects context across turns — not single-shot.
Model-Aware
Detects the base LLM and filters its default behaviors, so only system-prompt rules surface.
Self-Calibrating
Learns the user and assistant message selectors from a single marker message.
Auditable
Reports the reconstructed prompt plus a per-rule verdict: held, partial, or bypassed.
Three Moves

How it works.

01

Point

Give it a target — an OpenAI-compatible API, a generic HTTP endpoint, or a browser tab over CDP. It auto-starts Chrome and waits for your go.

02

Probe

The ReAct loop calibrates itself, then adapts each probe to what the agent just revealed — extracting persona, tools, constraints and behavior.

03

Reconstruct & Audit

Get a faithful reconstruction of the hidden system prompt, then an automatic robustness audit of the agent's own guardrails.

Quickstart
# probe a live browser tab — no presets, it self-calibrates
python3 -m reversegent --target-type browser \
  --browser-cdp-url http://localhost:9222 \
  --max-iterations 12 -o result.txt

# or an OpenAI-compatible API
python3 -m reversegent --target-type openai \
  --target-url https://api.example.com/v1 \
  --target-model my-agent -o result.txt
